React Kanban App

A Flux storage architecture is implemented using Alt.js. The changes in the application are stored in local storage to provide persistence.

Huffman Encoding

A program used to encode files with a large number (10 6) of integers. Implemented using 3 priority queues, namely Binary Heap, 4 Way Heap and Pairing Heap. Input file compressed to a bin file along with the code table. These files were then decoded to obtain the input file.

  • Source Code: Huffman Encoding
  • Date: March 2017
  • Technology Used: Java, Eclipse

P2P File Transfer

Program used to transfer a file using the peer-to-peer protocol. Information regarding the peers are read by the program and used to create a socket connection with the peers. The file is then broken into chunks and shared among all peers.
